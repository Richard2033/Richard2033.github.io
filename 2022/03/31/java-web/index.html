属性

location 指定上传文件存放的目录

maxRequestSize 指定上传文件的个数

Part接口

基本方法

方法
void delete 删除任何相关的临时磁盘文件
String getContentType() 获得客户端游览器设置的文件数据项的MIME类型
String getHeader(String name) 获得指定part头的一个字符串
InputStream getInputStream 获得输入流，通过这输入流来读取文件内容
String getName() 获得表单文件域的名称
long getSize() 获得文件的大小
void write(String fileName) 将文件上传到fileName指定的目录中
